# Wayfinders authored-map editor milestone

Status: implemented through `MAP-1.4` on 2026-07-21. This former proposal is no
longer a current design or planning authority.

Current ownership and dependency direction are in `ARCHITECTURE_MAP.md`.
Implemented definition, compiler, source, restart, editor, diagnostic, and
performance contracts are in `Wayfinders_Technical_Design.md`. The guarded map
repository transaction is in `Wayfinders_Asset_Pipeline.md`, and operator steps
are in `ASSET_PRODUCTION_QUICKSTART.md`. Durable milestone outcome, original
baseline signatures, deliberate exclusions, implementation references, and
acceptance coverage are archived in `Wayfinders_Roadmap_Archive.md`; volatile
gate, timing, browser, and sign-off state belongs only in
`IMPLEMENTATION_STATUS.md`.

The [MAP-1 editor concept](../concept_art/authored-map-editor/map-editor-milestone-concept.png)
remains a non-authoritative visual reference. The implemented Maps workspace is
a checked-in initial-world authoring tool, not a terrain painter, live-world
editor, or gameplay-session save/load system.
