# Great Hall navigator portrait prompts

These are the exact prompts used to create the twenty fixed navigator portraits
installed under `public/assets/gr5/great-hall/portraits`. Each portrait was
generated as a separate image using the same two visual references:

1. `../concept-a-refined-ancestor-wall.png`
2. `../infographics/03-navigator-plaque-states.png`

## Shared prompt

```text
Use case: stylized-concept
Asset type: Wayfinders Great Hall fixed navigator portrait
Input images: Image 1 is the selected Ancestor Wall material, palette, and pixel-art reference. Image 2 is the approved navigator plaque framing and portrait-scale reference.
Scene/backdrop: simple dark teal woven hall panel, quiet and unobtrusive.
Style/medium: crisp detailed pixel art matching the references; warm timber-age palette, turquoise, rust, sand sailcloth, shell accents; invented Wayfinders maritime visual language.
Composition/framing: exactly one front-facing bust portrait, head and shoulders centered, consistent player-facing portrait crop, generous margin, no surrounding plaque or frame.
Constraints: one unique fictional island navigator; practical woven sailcloth clothing; subtle shell or rope clasp; strong readable silhouette; no text, number, logo, watermark, weapon, crown, feathers, face paint, culturally identifiable sacred motif, fantasy rune, photorealism, or modern clothing.
Primary request: [navigator-specific request below]
```
