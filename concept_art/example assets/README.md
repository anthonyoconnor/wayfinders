# Wayfinders island and fishing-shoal candidates

Standalone visual candidates derived from the project design documents, concept art, and the inhabited-island brief. Each PNG contains exactly one asset. The set deliberately mixes uninhabited islands with inhabited variants belonging to visually distinct, long-separated tribes.

These files are source-art candidates only; they are not runtime-integrated and do not define collision, navigation, discovery state, or save identity. Static settlement structures may appear on inhabited variants, while people, boats, smoke, flags, fishing activity, and other changing elements remain separate future layers.

All PNGs are square `1254 × 1254` source renders with a consistent ocean field and one asset per image. The present prototype renderer uses a square orthographic navigation grid, while the approved concept-art direction uses a faux-isometric view. Runtime use therefore still requires reviewed masks, anchors, scale decisions, and renderer integration.

See `ASSET_INDEX.md` for the complete inventory.
